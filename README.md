# Calculadora-IMC
Desafio do Professor Edson Maia no YouTube de calculadora IMC 

Todas as imagens **tem autorização** para uso, foram retiradas do  [Unsplash](https://unsplash.com/) ou  do Google Imagens recortadas no Gimp.

Nesse desafio, foi incluido no código:
* estrutura HTML5 com: 
  *formulário
  *tabela
  *imagens
*CSS3 visando responsividade
*funcionalidade JavaScript com:
  *DOM
  *estrutura If
  *Functions
